Option Functions

sfBSCall

Calculates the price of a European call option using the Black-Scholes model.

Syntax: sfBSCall( S, K, T, r, q, sigma )

Arg Type Description
S float Spot price
K float Strike price
T float Time to maturity (years)
r float Risk-free rate
q float Continuous dividend yield
sigma float Volatility

Returns: float — Call option price. Returns Null if any parameter is non-positive.

Example:


price = sfBSCall( 100, 100, 1, 0.05, 0.02, 0.20 );

sfBSImpliedVol

Computes the implied volatility of a European option using Newton-Raphson iteration on the Black-Scholes model.

Syntax: sfBSImpliedVol( S, K, T, r, q, price, isCall )

Arg Type Description
S float Spot price
K float Strike price
T float Time to maturity (years)
r float Risk-free rate
q float Continuous dividend yield
price float Market price of the option
isCall float 1 for call, 0 for put

Returns: float — Implied volatility. Returns Null on invalid parameters.

Example:


iv = sfBSImpliedVol( 100, 100, 1, 0.05, 0.02, 10.45, 1 );

sfOptionPrice

Multi-asset dispatcher that prices options by underlying type using the appropriate model (Black-Scholes for spot equity / FX, Black-76 for futures / rate futures).

Syntax: sfOptionPrice( type, isCall, S_or_F, K, T, r, q, sigma )

Arg Type Description
type float Underlying type: 0 = Spot Equity, 1 = Future, 2 = FX, 3 = Rate Future
isCall float 1 for call, 0 for put
S_or_F float Spot or futures price
K float Strike price
T float Time to maturity (years)
r float Risk-free rate
q float Dividend yield (equity) or foreign rate (FX)
sigma float Volatility

Returns: float — Option price. Returns Null on invalid parameters.

Example:


// Equity call

p1 = sfOptionPrice( 0, 1, 100, 100, 1, 0.05, 0.02, 0.20 );

// Futures put

p2 = sfOptionPrice( 1, 0, 100, 100, 1, 0.05, 0, 0.20 );

sfImpliedVol

Multi-asset dispatcher that computes implied volatility by underlying type. Uses Newton-Raphson for futures and Brent's method for spot/FX.

Syntax: sfImpliedVol( type, isCall, S_or_F, K, T, r, q, price )

Arg Type Description
type float Underlying type: 0 = Spot Equity, 1 = Future, 2 = FX, 3 = Rate Future
isCall float 1 for call, 0 for put
S_or_F float Spot or futures price
K float Strike price
T float Time to maturity (years)
r float Risk-free rate
q float Dividend yield (equity) or foreign rate (FX)
price float Market price of the option

Returns: float — Implied volatility. Returns Null on invalid parameters.

Example:


// Equity call IV

iv1 = sfImpliedVol( 0, 1, 100, 100, 1, 0.05, 0.02, 10.45 );

// Futures put IV

iv2 = sfImpliedVol( 1, 0, 100, 100, 1, 0.05, 0, 8.91 );
