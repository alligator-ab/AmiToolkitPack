Queue Functions

Thread-safe FIFO message queues identified by string name. Queues are created

lazily on first use. All operations are thread-safe and can be used to exchange

messages between AFL threads.

sfQueuePushFloat

Pushes a float value to the end of the queue.

Syntax: sfQueuePushFloat( name, value )

Arg Type Description
name string Queue name
value float The float value to enqueue

Returns: Nothing

sfQueuePushString

Pushes a string value to the end of the queue (the string is copied).

Syntax: sfQueuePushString( name, str )

Arg Type Description
name string Queue name
str string The string value to enqueue

Returns: Nothing

sfQueuePop

Removes and returns the front element of the queue. If the queue is empty,

returns Null.

Syntax: sfQueuePop( name )

Arg Type Description
name string Queue name

Returns: AmiVar — The front value, or Null if empty.

sfQueuePeek

Returns the front element of the queue without removing it. If the queue is

empty, returns Null.

Syntax: sfQueuePeek( name )

Arg Type Description
name string Queue name

Returns: AmiVar — The front value, or Null if empty.

sfQueueSize

Returns the number of elements currently in the queue.

Syntax: sfQueueSize( name )

Arg Type Description
name string Queue name

Returns: float — Queue length.

sfQueueClear

Removes all elements from the queue and frees associated memory.

Syntax: sfQueueClear( name )

Arg Type Description
name string Queue name

Returns: Nothing

sfQueueExists

Checks whether a queue with the given name exists (has been created by a

previous push/pop/size/clear operation).

Syntax: sfQueueExists( name )

Arg Type Description
name string Queue name

Returns: float — 1 if the queue exists, 0 otherwise.

Queue Example


// Push values

sfQueuePushFloat("prices", 100.0);

sfQueuePushFloat("prices", 101.5);

sfQueuePushString("prices", "note");



// Check size

size = sfQueueSize("prices"); // 3



// Peek at front (without removing)

front = sfQueuePeek("prices"); // 100.0



// Pop in FIFO order

v1 = sfQueuePop("prices"); // 100.0

v2 = sfQueuePop("prices"); // 101.5

v3 = sfQueuePop("prices"); // "note"



// Pop from empty queue returns Null

v4 = sfQueuePop("prices"); // Null (IsNull(v4) == 1)



// Clear

sfQueuePushFloat("prices", 1.0);

sfQueueClear("prices");



// Check existence

exists = sfQueueExists("prices"); // 1 (was just used)
