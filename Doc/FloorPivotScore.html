Floor Pivot Score

The sfFloorPivotScore function quantifies the relative position of a price

within the Floor Pivot level framework (Pivot, R1-R3, S1-S3). It computes

consecutive non-overlapping periods from a StartPeriod marker array, uses each

completed period's High/Low/Close to compute Floor Pivot levels, then assigns a

linear score from −3 (below S3) to +3 (above R3) to every bar of the following

period.

Syntax: sfFloorPivotScore( Price, High, Low, Close, StartPeriod )

Arg Type Description
Price array Price series to score (e.g. Close)
High array High prices for pivot calculation
Low array Low prices for pivot calculation
Close array Close prices (last close of each period = CC)
StartPeriod array 1 on the first bar of each period, 0 elsewhere

Returns: array of scores clamped to [−3, +3]

Scoring zones:

Zone Score range
S3 → S2 −3 to −2 (linear)
S2 → S1 −2 to −1
S1 → Pivot −1 to 0
Pivot 0
Pivot → R1 0 to 1
R1 → R2 1 to 2
R2 → R3 2 to 3
< S3 / > R3 −3 / +3 (clamped)

Example:


StartPeriod[0] = 1; StartPeriod[10] = 1; StartPeriod[20] = 1;

Score = sfFloorPivotScore( Close, High, Low, Close, StartPeriod );

Notes:

  • Requires at least two complete periods (i.e. at least two `1` markers)

    • before scoring begins. Bars in the first period and bars before the first

    marker return Empty.

  • Levels are computed from the preceding completed period:

    • HH = highest High, LL = lowest Low, CC = last Close of that period.

  • Periods are disjoint — the next period's levels are ignored while scoring

    • the current one.
