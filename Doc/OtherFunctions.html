Other Functions

Internal / diagnostic functions that expose AmiBroker's plugin site interface,

memory management, and variable access to AFL. Use with care.

sfGetPluginInfo

Returns a formatted string with plugin metadata (name, vendor, version, type, etc.)

by calling GetPluginInfo on the plugin interface.

Syntax: sfGetPluginInfo()

Returns: string — e.g. Name:"AmiToolkit Plug-in",Vendor:"SF",Version:"1.0.0",IDCode:0,Type:1,...

sfCheckArrayInit

Checks whether an array is initialized (all elements equal) and describes its state.

Syntax: sfCheckArrayInit( Array )

Arg Type Description
Array array The array to inspect

Returns: string — A description of the array state:

  • `"Array is allocated and initialized with a single value ..."` if all elements are identical
  • `"Array is allocated but not initialized - several values detected"` if elements differ
  • `"Error: Wrong Argument: Array expected"` if the argument is not an array

    • sfInitArray

    Quickly fills an array with a single float value using an optimised

    write-once + doubling (memmove) algorithm, with AVX acceleration

    when available.

    Syntax: sfInitArray( Array [, value ] )

    Arg Type Description
    Array array Array to initialise (modified in-place)
    value float (optional) Fill value. Default: Empty

    Returns: float — 0 on success, -1 on error (wrong argument types).

    Notes:

  • Uses `memmove` doubling (O(log n) steps) for bulk filling.
  • If the CPU supports AVX, a seed-and-AVX strategy is used for arrays ≥ 256 elements.
  • The array is modified in-place; no new allocation is made.

    • sfIsAllocArrayInit

    Allocates a new array result via AllocArrayResult and immediately checks

    its initialisation state via CheckArrayInit. Useful for testing whether

    newly allocated arrays are pre-initialised.

    Syntax: sfIsAllocArrayInit()

    Returns: string — Same as CheckArrayInit applied to a freshly allocated array.

    sfEmptyValue

    Returns the EMPTY_VAL constant used by AmiBroker to represent missing data.

    Syntax: sfEmptyValue()

    Returns: float — The EMPTY_VAL sentinel value.

    sfEqualVariables

    Checks whether two AFL variables refer to the same underlying object

    (identity check, not value equality). Works for float, array, string,

    and display-pointer types.

    Syntax: sfEqualVariables( VarName, VarValue )

    Arg Type Description
    VarName string Name of an existing AFL variable
    VarValue AmiVar A value or variable to compare against

    Returns: float — 1 if both refer to the same object, 0 otherwise.

    Note: For arrays this checks pointer identity, not element-by-element equality.

    sfAlloc

    Allocates a raw memory buffer through AmiBroker's site interface (gSite.Alloc).

    Syntax: sfAlloc( size )

    Arg Type Description
    size float Number of bytes to allocate

    Returns: disp — A display pointer to the allocated memory, or Nothing if allocation fails.

    Warning: Memory allocated with sfAlloc must be freed with sfFree.

    Use only for interop with AmiBroker's memory manager.

    sfFree

    Frees a memory buffer previously allocated with sfAlloc.

    Syntax: sfFree( pointer )

    Arg Type Description
    pointer disp Display pointer returned by sfAlloc

    Returns: float — 1 on success.

    sfAllocArrayResult

    Allocates a new AFL array result via gSite.AllocArrayResult. The array

    is pre-filled with EMPTY_VAL.

    Syntax: sfAllocArrayResult( size )

    Arg Type Description
    size float Number of elements

    Returns: array — A new array initialised to EMPTY_VAL.

    sfGetArrayLength

    Returns the length of the initialised (non-empty) prefix of an array

    using binary search. Useful for arrays that were allocated larger than

    the data they contain.

    Syntax: sfGetArrayLength( Array [, maxSize ] )

    Arg Type Description
    Array array The array to measure
    maxSize float (optional) Maximum size to consider. Default: full array size

    Returns: float — The number of leading non-Empty elements.

    sfGetVariable

    Retrieves an AFL variable by name via gSite.GetVariable.

    Syntax: sfGetVariable( name )

    Arg Type Description
    name string The variable name

    Returns: AmiVar — The variable's value (float, string, array, or Nothing).

    sfSetArray

    Sets an AFL variable to an array value via gSite.SetVariable.

    Syntax: sfSetArray( name, array )

    Arg Type Description
    name string Variable name to create / overwrite
    array array The array value to assign

    Returns: float — 1 on success.

    sfSetFloat

    Sets an AFL variable to a float value via gSite.SetVariable.

    Syntax: sfSetFloat( name, value )

    Arg Type Description
    name string Variable name to create / overwrite
    value float The float value to assign

    Returns: float — 1 on success.

    sfArrayShiftLeft

    Shifts all non-empty elements of an array to the left, filling the vacated

    positions at the right with EMPTY_VAL.

    Syntax: sfArrayShiftLeft( Array )

    Arg Type Description
    Array array Input array

    Returns: array — A new array with elements shifted left.

    Example:

    
// a = [Empty, 10, 20, 30, Empty];

b = sfArrayShiftLeft(a);  // [10, 20, 30, Empty, Empty]

    sfArrayShiftRight

    Shifts all non-empty elements of an array to the right, filling the vacated

    positions at the left with EMPTY_VAL.

    Syntax: sfArrayShiftRight( Array )

    Arg Type Description
    Array array Input array

    Returns: array — A new array with elements shifted right.

    Example:

    
// a = [10, 20, 30, Empty, Empty];

b = sfArrayShiftRight(a);  // [Empty, Empty, 10, 20, 30]
