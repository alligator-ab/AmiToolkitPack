Equity Valuation Functions

sfWACC

Calculates the Weighted Average Cost of Capital.

Syntax: sfWACC( equity, debt, costEquity, costDebt, taxRate )

Arg Type Description
equity float Market value of equity
debt float Market value of debt
costEquity float Cost of equity (e.g. from CAPM)
costDebt float Pre-tax cost of debt
taxRate float Corporate tax rate (e.g. 0.21 for 21 %)

Formula:


WACC = (E / V) × Ke + (D / V) × Kd × (1 − T)

where V = E + D.

Returns: float — The WACC as a decimal. Returns Null if parameters are invalid (negative values, taxRate outside [0,1), or zero total firm value).

Example:


wacc = sfWACC( 500, 300, 0.10, 0.05, 0.21 );

// E=500, D=300, Ke=10%, Kd=5%, tax=21%

sfTerminalValue

Calculates the Terminal Value (continuing value) using the Gordon Growth Model.

Syntax: sfTerminalValue( fcf, wacc, growthRate )

Arg Type Description
fcf float Free Cash Flow of the last projected year
wacc float Weighted Average Cost of Capital (discount rate)
growthRate float Perpetual growth rate (must be less than wacc)

Formula:


TV = FCF × (1 + g) / (WACC − g)

Returns: float — The terminal value. Returns Null if any argument is negative, if wacc <= 0, or if growthRate >= wacc (non-convergent).

Example:


// 5% WACC, 2% perpetual growth, 100M FCF

tv = sfTerminalValue( 100, 0.05, 0.02 );
