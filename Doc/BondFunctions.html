Bond Functions

sfBondPrice

Calculates the theoretical price of a bond for a given yield to maturity.

Syntax: sfBondPrice( nominal, couponRate, years, paymentsPerYear, yield )

Arg Type Description
nominal float Face value of the bond
couponRate float Annual coupon rate (e.g. 0.05 for 5 %)
years float Time to maturity in years
paymentsPerYear float Number of coupon payments per year (e.g. 2 for semi‑annual)
yield float Yield to maturity (annual, e.g. 0.05 for 5 %)

Returns: float — The theoretical bond price. Returns Null if required arguments are missing or parameters are invalid (non‑positive nominal, years, or payments per year).

Formula:


P = Σ C/(1+Y/f)^t + N/(1+Y/f)^n

where C = coupon per period, Y = annual yield, f = payments/year, n = total payments, N = nominal.

Example:


// 5-year bond, 4% semi-annual coupon, 3% yield

price = sfBondPrice( 100, 0.04, 5, 2, 0.03 );

sfYieldToMaturity

Calculates the yield to maturity of a bond given its market price, using Newton‑Raphson iteration.

Syntax: sfYieldToMaturity( nominal, couponRate, years, paymentsPerYear, marketPrice )

Arg Type Description
nominal float Face value of the bond
couponRate float Annual coupon rate (e.g. 0.05 for 5 %)
years float Time to maturity in years
paymentsPerYear float Number of coupon payments per year (e.g. 2 for semi‑annual)
marketPrice float Current market price of the bond

Returns: float — The annual yield to maturity, or Null on invalid arguments. Returns NaN if Newton‑Raphson does not converge within 100 iterations.

Algorithm:

1. Start with an initial guess of 5 %.

2. Iterate y_new = y - (P(y) - marketPrice) / P'(y) using a numerical derivative.

3. Converge when |P(y) - marketPrice| < 1e-8.

Example:


// 5-year bond, 4% semi-annual coupon, price = 104.58

ytm = sfYieldToMaturity( 100, 0.04, 5, 2, 104.58 );

sfBondDV01

Calculates the Dollar Value of a 1 basis point (DV01) move for a bond, using the analytical first derivative of price with respect to yield.

Syntax: sfBondDV01( nominal, couponRate, years, paymentsPerYear, yield )

Arg Type Description
nominal float Face value of the bond
couponRate float Annual coupon rate (e.g. 0.05 for 5 %)
years float Time to maturity in years
paymentsPerYear float Number of coupon payments per year (e.g. 2 for semi‑annual)
yield float Yield to maturity (annual)

Formula (analytical):


DV01 = [ Σ t·C_f/(1+y/f)^(t+1) + n·N/(1+y/f)^(n+1) ] / (f · 10000)

where C_f = N·c/f, n = years·f, r = y/f.

Returns: float — DV01 (price change for +1bp in yield). Returns Null on invalid parameters.

Example:


// 5-year, 5% semi-annual coupon, 5% yield

dv01 = sfBondDV01( 100, 0.05, 5, 2, 0.05 );

sfFutureDV01

Calculates the DV01 of a Treasury bond futures contract using the desk‑ready CTD conversion method: FutDV01 = BondDV01(CTD) / CF.

Syntax: sfFutureDV01( ctdNominal, ctdCouponRate, ctdYears, ctdPayPerYear, ctdYield, convFactor )

Arg Type Description
ctdNominal float Face value of the cheapest‑to‑deliver bond
ctdCouponRate float Annual coupon rate of the CTD
ctdYears float Years to maturity of the CTD
ctdPayPerYear float Coupon frequency of the CTD
ctdYield float Yield to maturity of the CTD
convFactor float Conversion factor of the CTD for the futures contract

Returns: float — Futures DV01. Returns Null on invalid parameters or if convFactor ≤ 0.

Example:


// CTD: 100 face, 6% coupon, 15yr, semi-annual, 5% yield, CF=1.05

futDV01 = sfFutureDV01( 100, 0.06, 15, 2, 0.05, 1.05 );

sfEuroBondDV01

Convenience wrapper for sfBondDV01 with annual payments (European bond convention).

Syntax: sfEuroBondDV01( nominal, couponRate, years, yield )

Arg Type Description
nominal float Face value
couponRate float Annual coupon rate
years float Years to maturity
yield float Yield to maturity

Returns: float — DV01 for 1bp at annual frequency.

Example:


dv01 = sfEuroBondDV01( 100, 0.04, 10, 0.03 );

sfEuroFutureDV01

Convenience wrapper for sfFutureDV01 with annual coupon frequency (European bond futures convention).

Syntax: sfEuroFutureDV01( ctdNominal, ctdCouponRate, ctdYears, ctdYield, convFactor )

Arg Type Description
ctdNominal float Face value of the CTD
ctdCouponRate float Annual coupon rate of the CTD
ctdYears float Years to maturity of the CTD
ctdYield float Yield to maturity of the CTD
convFactor float Conversion factor of the CTD

Returns: float — Futures DV01 (annual frequency). Returns Null on invalid parameters.

Example:


futDV01 = sfEuroFutureDV01( 100, 0.05, 10, 0.04, 1.02 );
